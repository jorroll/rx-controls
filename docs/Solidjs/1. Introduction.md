# Introduction for Solidjs users

`rx-controls-solid` aims to simplify the process of dealing with forms and form validation in your Solidjs apps.

For example:

```tsx
const TextInput = withControl((props) => {
  const control = props.control;

  return (
    <input
      type="text"
      value={control.value}
      oninput={(e) => {
        control.markDirty(true);
        control.setValue(e.currentTarget.value || null);
      }}
      onblur={() => control.markTouched(true)}
      ref={props.ref}
      placeholder={props.placeholder}
    />
  );
});

const PersonNameForm = withControl({
  // here we provide a function which creates the control we want our
  // component to have
  controlFactory: () =>
    new FormGroup({
      firstName: new FormControl(null),
      lastName: new FormControl(null),
    }),
  // here we define our component
  component: (props) => {
    const control = props.control;

    const onFormSubmit = (e) => {
      e.preventDefault();

      const formValue = control.value;
      // do stuff
    };

    return (
      <form onsubmit={onFormSubmit}>
        <TextInput controlName="firstName" />
        <TextInput controlName="lastName" />
      </form>
    );
  },
});
```
